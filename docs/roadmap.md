Not promises - genuine goals:

| Feature | Notes |
|---|---|
| **Browser client** | Use Starl without installing anything |
| **Desktop client** | Native desktop app |
| **Manual downloads** | Save songs explicitly for offline use |
| **Radio stations** | Auto-generated based on genres and moods |
| **Video support** | Music videos in the same interface |
| **Plugin system** | Community-hosted plugins |
| **Full settings panel** | Crossfade, gapless, normalize, privacy, accessibility, storage |
| **Private session mode** | Listen without saving history |
| **Sleep timer** | Stop after a song, playlist, or time limit |
| **Home screen widgets** | Display currently playing song or quick actions on the home screen. |
| **Android Material palette** | Use the Android Material Design color palette natively as a theme |
| **Download tracks** | Manually download tracks to your device (not cache, actual files) |
