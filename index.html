FAQ

Is Starl open-source?
Yes — Starl is released under the PFF License ( PURE FREEDOM FOREVER ).
Does it collect telemetry?
No telemetry is collected. The app may optionally integrate with services you enable (e.g., Discord) and those integrations are under your control.
How do I run the Starl Server?
Check the Docs for a step-by-step guide. In short: run the server binary, point the app to your server address, and enable synchronization in settings.
How do Live Lyrics work?
Live lyrics are matched to tracks and synced in real time. You can download lyric packs for offline use. Accuracy improves with local caches.
How can I contribute or report bugs?
Open issues or pull requests on the GitHub repo. For security-sensitive bugs, contact the maintainers privately (see Docs).
